Notes of Github

//////////////////////////////////////

# git init
# description: Initializes a new Git repository, creating a .git subdirectory with configuration files.

//////////////////////////////////////

# git status
# description: Displays the status of changes as untracked, modified, or staged in the working directory.

//////////////////////////////////////

# git add
# description: Adds changes in the working directory to the staging area, preparing them for a commit.

//////////////////////////////////////

# git diff
# description: Shows the differences between the working directory and the staging area or the last commit.

//////////////////////////////////////

# git commit
# description: Records changes in the working directory to the local repository, creating a new commit with a unique identifier.

//////////////////////////////////////
